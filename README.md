# 基于Vue的聊天室实现

使用了pusher的chatkit插件和管理，注册需要在后台由管理员添加用户，权限可由管理员在pusher/chatkit提供的后台进行操作

## 项目安装
```
npm install
```

### 编译和热加载以用于开发模式
```
npm run serve
```

### 编译和压缩以用于生产模式
```
npm run build
```
